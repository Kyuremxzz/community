ASCIIFX — DEMO DA ENGINE

TaskForge · engine visual ASCII estilo rubber-hose. Cada secao exercita uma funcao da API.

AsciiFX.title(text)

Banner figlet gerado como string (nao anima, nao toca DOM — o script injeta via textContent).
